1. 함수의 기본 (60분)
1.1. 매개변수와 반환 타입 지정 (20분)
목표: 함수의 매개변수와 반환 타입을 명시적으로 선언하는 방법 학습
내용:
함수 선언 및 타입 지정 방법
실습: 간단한 문자열 처리 함수 및 숫자 연산 함수 작성
1.2. 선택적 매개변수와 기본 매개변수 (40분)
목표: 선택적 매개변수와 기본 매개변수의 사용법 이해
내용:
선택적 매개변수(?)와 기본 매개변수의 구문 및 사용 시나리오
실습: 사용자 정보를 받아 인사를 출력하는 함수 작성(이름은 필수, 나이는 선택적)


2. 함수 오버로딩 (60분)
2.1. 함수 오버로딩의 기본 (30분)
목표: 하나의 함수 이름으로 여러 가지 매개변수 타입과 반환 타입을 처리할 수 있는 방법 학습
내용:
함수 오버로딩의 개념과 구현 방법
실습: 다양한 타입을 입력으로 받아 처리하는 getInfo 함수 오버로딩 구현
2.2. 함수 오버로딩의 활용 (30분)
목표: 실제 사용 사례를 통해 함수 오버로딩의 필요성과 장점 이해
내용:
실제 코드에서 함수 오버로딩을 활용하는 사례 분석
실습: API 호출의 결과를 다루는 함수 오버로딩 구현


3. 화살표 함수와 this 컨텍스트 (60분)
3.1. 화살표 함수의 특징 (30분)
목표: 화살표 함수의 구문, 특징 및 사용법 이해
내용:
화살표 함수의 기본 구문과 특징 설명
this 바인딩의 차이점 설명
실습: 화살표 함수를 사용하여 리스트의 각 요소에 작업을 적용하는 함수 작성
3.2. 화살표 함수와 this 컨텍스트의 이해 (30분)
목표: 화살표 함수에서의 this 행동 방식을 이해하고 일반 함수와의 차이점 학습
내용:
화살표 함수에서 this가 어떻게 작동하는지 설명
실습: 클래스 내부에서 화살표 함수와 일반 함수를 사용하여 this의 차이를 체험


4. 함수와 인터페이스 (30분)
4.1. 인터페이스를 활용한 함수 타이핑 (15분)
목표: 인터페이스를 사용하여 함수 매개변수와 반환 타입을 보다 구조적으로 정의하는 방법 학습
내용:
인터페이스를 사용한 함수 매개변수의 타이핑 예제
실습: 주어진 인터페이스를 준수하는 함수 작성
4.2. 인터페이스의 실용적 활용 (15분)
목표: 인터페이스를 실제 프로젝트에서 효과적으로 사용하는 방법 이해
내용:
실제 프로젝트에서 인터페이스가 코드 관리와 확장에 어떤 이점을 제공하는지 사례 연구
실습: 인터페이스를 활용하여 데이터 처리 함수 집합 구현
